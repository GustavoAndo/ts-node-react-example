# ts-node-react-example

## Como rodar o código:

### Iniciando o node:
1) Abra o terminal.
2) Entre na pasta "back":  `cd back`
3) Instale as dependências:  `npm install`
4) Inicie o server: `npm run dev`

### Iniciando o react:
1) Abra um novo terminal.
2) Entre na pasta "front":  `cd front`
3) Instale as dependências:  `npm install`
4) Inicie o projeto:  `npm start`
